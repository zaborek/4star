1. make sure "TAP001.TXT" and cards.r are in the same folder
2. run cards.r to produce cards.rds and barcodes.txt.
cards.rds contains the cleaned title data, barcodes.txt contains a list of barcodes for python to use.
3. use python3 to run barcode_gen.py to generate new barcodes
4. run cardprint_update.r to make new pdfs and update old pdfs
